# Online Recruitment System
A data-driven recruitment analytics platform designed to optimize hiring workflows, improve candidate screening efficiency, and provide actionable insights into recruitment pipelines.

## Problem Statement
Traditional recruitment processes are inefficient, manual, and lack data-driven decision-making. 
This project addresses challenges such as:
- Inefficient candidate screening
- Lack of visibility into hiring pipelines
- Poor data organization across recruitment stages

## Solution
Developed an end-to-end recruitment analytics system that:
- Processes candidate and job data
- Models recruitment workflows using structured data
- Generates insights for hiring decisions
- Visualizes recruitment performance metrics

## Tech Stack
- Python (Pandas, NumPy)
- SQL (Data Modeling & Queries)
- Excel (Data Source)
- Data Visualization (Matplotlib / Power BI / Tableau)

## System Architecture
1. Data Ingestion (Excel datasets)
2. Data Cleaning & Transformation
3. SQL-based Data Modeling
4. Analytical Processing
5. Visualization & Insights

## Key Features
- Candidate Data Management
- Recruitment Funnel Analysis
- Hiring Trend Insights
- Data Cleaning & Preprocessing Pipeline
- SQL-based relational data model
